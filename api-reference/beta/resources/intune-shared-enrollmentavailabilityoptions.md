# <a name="enrollmentavailabilityoptions-enum-type"></a>tipo de número enrollmentAvailabilityOptions

Namespace: microsoft.graph

> **Importante:** As APIs Graph Microsoft na versão /beta estão sujeitas a alterações; não há suporte para uso de produção.

> **Observação:** A API do Microsoft Graph para Intune requer uma [licença ativa do Intune](https://go.microsoft.com/fwlink/?linkid=839381) para o locatário.

Opções disponíveis para personalização de fluxo de registro

## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|availableWithPrompts|0|O fluxo de registro do dispositivo é mostrado para o usuário final com prompts de registro guiados|
|availableWithoutPrompts|1|O fluxo de registro do dispositivo está disponível para o usuário final sem prompts de registro guiados|
|unavailable|2|O fluxo de registro do dispositivo não está disponível para o usuário final|
