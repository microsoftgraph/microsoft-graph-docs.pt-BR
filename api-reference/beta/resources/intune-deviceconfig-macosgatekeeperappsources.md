Namespace: microsoft.graph

> **Importante:** As APIs Graph Microsoft na versão /beta estão sujeitas a alterações; não há suporte para uso de produção.

> **Observação:** A API do Microsoft Graph para Intune requer uma [licença ativa do Intune](https://go.microsoft.com/fwlink/?linkid=839381) para o locatário.

Opções de origem do aplicativo para o gatekeeper do macOS.

## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|notConfigured|0|Valor padrão do dispositivo, sem intenção.|
|macAppStore|1|Somente aplicativos do Mac AppStore podem ser executados.|
|macAppStoreAndIdentifiedDevelopers|2|Somente aplicativos do Mac AppStore e desenvolvedores identificados podem ser executados.|
|em qualquer lugar|3|Aplicativos de qualquer lugar podem ser executados.|
