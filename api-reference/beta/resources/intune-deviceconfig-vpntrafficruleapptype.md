> **Observação:** A API do Microsoft Graph para Intune requer uma [licença ativa do Intune](https://go.microsoft.com/fwlink/?linkid=839381) para o locatário.

Indica o tipo de aplicativo ao qual uma regra de tráfego VPN está associada.

|Membro|Valor|Descrição|
|:---|:---|:---|
|Nenhuma|0|A regra de tráfego não está associada a um aplicativo.|
|desktop|1 |A regra de tráfego está associada a um aplicativo da área de trabalho.|
|universal|2|A regra de tráfego está associada a um aplicativo Universal.|
