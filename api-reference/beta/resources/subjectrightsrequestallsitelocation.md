# <a name="subjectrightsrequestallsitelocation-resource-type"></a>Tipo de recurso subjectRightsRequestAllSiteLocation

Namespace: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Representa as propriedades de uma solicitação de direitos de entidade que define todos os sites (SharePoint sites, OneDrive for Business sites e Microsoft Teams canais) como um local de pesquisa. Use esse tipo quando quiser procurar dados em todos os sites.

Herda de [subjectRightsRequestSiteLocation](../resources/subjectrightsrequestsitelocation.md).

## <a name="properties"></a>Propriedades
Nenhum.
