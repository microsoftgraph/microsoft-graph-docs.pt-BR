# <a name="removeaccessapplyaction-resource-type"></a>tipo de recurso removeAccessApplyAction

Namespace: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

[!INCLUDE [accessreviews-disclaimer-v2](../../includes/accessreviews-disclaimer-v2.md)]

Remova o acesso de um recurso para aqueles negados em uma revisão de acesso desse recurso. Essa é a opção padrão para[accessReviewApplyAction](../resources/accessreviewapplyaction.md) se uma opção não for passada.

Herda de [accessReviewApplyAction](../resources/accessreviewapplyaction.md).

Nenhum.
