# <a name="mobileappdependencytype-enum-type"></a>Tipo de número mobileAppDependencyType

Namespace: microsoft.graph

> **Importante:** As APIs Graph Microsoft na versão /beta estão sujeitas a alterações; não há suporte para uso de produção.

> **Observação:** A API do Microsoft Graph para Intune requer uma [licença ativa do Intune](https://go.microsoft.com/fwlink/?linkid=839381) para o locatário.

Indica o tipo de dependência associado a uma relação entre dois aplicativos móveis.

## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|detect|0|Indica que o aplicativo filho deve ser detectado antes de instalar o aplicativo pai.|
|autoInstall|1 |Indica que o aplicativo filho deve ser instalado antes de instalar o aplicativo pai.|
