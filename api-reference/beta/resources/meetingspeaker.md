# <a name="meetingspeaker-resource-type"></a>Tipo de recurso meetingSpeaker

Namespace: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Representa o orador de uma reunião online que tem o registro habilitado.

## <a name="properties"></a>Propriedades

| Propriedade | Tipo | Descrição |
| :------- | :--- | :---------- |
| bio | Cadeia de caracteres | Bio do alto-falante. |
| displayName | Cadeia de caracteres | Nome de exibição do alto-falante. |
