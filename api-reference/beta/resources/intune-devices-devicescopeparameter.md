# <a name="devicescopeparameter-enum-type"></a>Tipo de enumeração deviceScopeParameter

Namespace: microsoft.graph

> **Importante:** As APIs Graph Microsoft na versão /beta estão sujeitas a alterações; não há suporte para uso em produção.

> **Observação:** A API do Microsoft Graph para Intune requer uma [licença ativa do Intune](https://go.microsoft.com/fwlink/?linkid=839381) para o locatário.

Parâmetro de configuração do escopo do dispositivo. Ele será gasto no futuro para adicionar mais parâmetros. Por exemplo: o parâmetro de escopo do dispositivo pode ser versão do sistema operacional, Tipo de Disco, Fabricante do dispositivo, modelo de dispositivo ou marca de escopo. Valor padrão: scopeTag.

## <a name="members"></a>Membros
|Membro|Valor|Descrição|
|:---|:---|:---|
|none|0|O parâmetro De escopo do dispositivo não está definido|
|scopeTag|1|use a Marca de Escopo como parâmetro para a configuração do escopo do dispositivo.|
|unknownFutureValue|2|Valor de espaço reservado para expansão futura.|
