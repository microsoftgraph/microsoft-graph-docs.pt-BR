## <a name="properties"></a>Propriedades
|Propriedade|Tipo|Descrição|
|:---|:---|:---|
|id|Cadeia de caracteres|Chave da entidade.|
|targetGroupId|Cadeia de caracteres|A ID do grupo do AAD para o qual estamos direcionando a configuração do dispositivo.|
|excludeGroup|Booliano|Indica se esse grupo deve ser excluído. Padrões de que o grupo deve ser incluído|

## <a name="relationships"></a>Relações
|Relação|Tipo|Descrição|
|:---|:---|:---|
