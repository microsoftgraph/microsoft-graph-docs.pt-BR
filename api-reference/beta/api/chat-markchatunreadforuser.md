# <a name="chat-markchatunreadforuser"></a>chat: markChatUnreadForUser
Namespace: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Marque um [chat](../resources/chat.md) como não lido para um usuário.

## <a name="permissions"></a>Permissões
Uma das seguintes permissões é obrigatória para chamar esta API. Para saber mais, incluindo como escolher permissões, confira [Permissões](/graph/permissions-reference).

|Tipo de permissão|Permissões (da com menos para a com mais privilégios)|
|:---|:---|
|Delegado (conta corporativa ou de estudante)|Chat.ReadWrite|
|Delegado (conta pessoal da Microsoft) | Sem suporte. |
|Aplicativo | Sem suporte. |

## <a name="http-request"></a>Solicitação HTTP

<!-- {
  "blockType": "ignored"
}
-->
``` http
POST /chats/{chat-id}/markChatUnreadForUser
```

## <a name="request-headers"></a>Cabeçalhos de solicitação
|Nome|Descrição|
|:---|:---|
|Autorização|{token} de portador. Obrigatório.|
|Content-Type|application/json. Obrigatório.|

## <a name="request-body"></a>Corpo da solicitação
No corpo da solicitação, fornece uma representação JSON dos parâmetros.

A tabela a seguir mostra os parâmetros que podem ser usados com esta ação.

|Parâmetro|Tipo|Descrição|
|:---|:---|:---|
|lastMessageReadDateTime|DateTimeOffset|Timestamp que indica o tempo após o qual todas as mensagens enviadas ou recebidas devem ser marcadas como não lidas.|
|tenantId|String|ID de locatário do usuário.|
|usuário|[teamworkUserIdentity](../resources/teamworkuseridentity.md)|Usuário para desinteressar o chat.|

> **Observação:** **lastMessageReadDateTime** é opcional na solicitação. Se não for mencionado, a última mensagem será marcada como não lida.

## <a name="response"></a>Resposta

Se tiver êxito, esta ação retornará um código de resposta `204 No Content`.
