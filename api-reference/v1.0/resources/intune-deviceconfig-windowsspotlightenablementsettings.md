# <a name="windowsspotlightenablementsettings-enum-type"></a>Tipo de enumeração windowsSpotlightEnablementSettings

Namespace: microsoft.graph

> **Observação:** A API do Microsoft Graph para Intune requer uma [licença ativa do Intune](https://go.microsoft.com/fwlink/?linkid=839381) para o locatário.

Permite que o administrador de TI defina um mecanismo de pesquisa padrão predefinido para MDM-Controlled dispositivos
