# <a name="teamvisibilitytype-enum-type"></a>Tipo denum teamVisibilityType



Descreve a visibilidade de uma [equipe](../resources/team.md). 

## <a name="members"></a>Membros

| Membro | Valor| Descrição |
|:---------------|:--------|:----------|
|private|0|Qualquer pessoa pode ver a equipe, mas somente o proprietário pode adicionar um usuário à equipe.|
|public|1|Qualquer pessoa pode ingressar na equipe.|
