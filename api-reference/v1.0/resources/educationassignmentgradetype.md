# <a name="educationassignmentgradetype-resource-type"></a>Tipo de recurso educationAssignmentGradeType

Namespace: microsoft.graph

Representa como uma atribuição será gradeada.

Isso é usado na **propriedade assignments.grading.** Essa superclasse não pode ser usada diretamente na propriedade assignment. Ela existe para permitir várias maneiras diferentes de fazer atribuições de classificação.


## <a name="properties"></a>Propriedades

Nenhum.
