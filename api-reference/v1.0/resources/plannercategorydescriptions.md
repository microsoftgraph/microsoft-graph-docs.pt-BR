# <a name="plannercategorydescriptions-resource-type"></a>Tipo de recurso plannerCategoryDescriptions

Namespace: microsoft.graph

Representa os rótulos descritivos para as categorias que foram definidas para um plano. Ele pertence ao objeto [de detalhes do](plannerplandetails.md) plano. Pode haver até 25 categorias definidas.
