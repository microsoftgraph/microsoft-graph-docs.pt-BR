# <a name="targetapplicationowners-resource-type"></a>Tipo de recurso targetApplicationOwners

Namespace: microsoft.graph

Usado em uma política de atribuição de pacote de acesso, esse tipo herda de [subjectSet](../resources/subjectset.md) e indica que os proprietários de aplicativos podem solicitar um pacote de acesso em nome desse aplicativo.


## <a name="properties"></a>Propriedades

Nenhum.
## <a name="relationships"></a>Relações
Nenhum
