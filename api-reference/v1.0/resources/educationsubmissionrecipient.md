# <a name="educationsubmissionrecipient-resource-type"></a>Tipo de recurso educationSubmissionRecipient

Namespace: microsoft.graph

Classe abstrata que representa os diferentes conjuntos de usuários aos quais um envio pode ser atribuído. 

Os valores possíveis incluem indivíduos ou grupos. Isso é usado na **propriedade submission.recipient.**


## <a name="properties"></a>Propriedades
Nenhum.
