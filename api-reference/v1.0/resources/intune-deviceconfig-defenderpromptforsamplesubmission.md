|Membro|Valor|Descrição|
|:---|:---|:---|

|promptBeforeSendingPersonalData|2|Envie amostras seguras automaticamente.|
|neverSendData|3|Nunca envie dados.|
|sendAllDataWithoutPrompting|4 |Envie todos os dados sem solicitar.|
