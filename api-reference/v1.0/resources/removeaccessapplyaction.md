# <a name="removeaccessapplyaction-resource-type"></a>tipo de recurso removeAccessApplyAction

Namespace: microsoft.graph

Remove o acesso a um recurso para aqueles negados em uma revisão de acesso desse recurso. Essa é a opção padrão escolhida para [accessReviewApplyAction](../resources/accessreviewapplyaction.md) se uma opção não for passada.

Herda de [accessReviewApplyAction](../resources/accessreviewapplyaction.md).

## <a name="properties"></a>Propriedades
Nenhum.

## <a name="relationships"></a>Relações
Nenhum
