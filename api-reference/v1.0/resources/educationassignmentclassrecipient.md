# <a name="educationassignmentclassrecipient-resource-type"></a>tipo de recurso educationAssignmentClassRecipient

Namespace: microsoft.graph

Tipo de recurso usado dentro da [propriedade assignment.assignTo.](educationassignment.md) 

Quando definido como destinatário de classe, todos os alunos da classe receberão um objeto de envio quando a atribuição for publicada.

Esse recurso é uma subclasse [de educationAssignmentRecipient](educationassignmentrecipient.md).

## <a name="properties"></a>Propriedades

Nenhum.
